# こんにちは表示プログラム

## ダウンロードありがとうございます！

## 概要
このプログラムは、コンソールに「こんにちは」と表示するシンプルなPythonプログラムです。

## 機能
- コンソールに「こんにちは」と表示します
- シンプルで分かりやすいコード

## 使い方
1. Pythonがインストールされていることを確認してください
2. コマンドプロンプトやターミナルを開きます
3. 以下のコマンドを実行します：
   ```
   python hello.py
   ```
